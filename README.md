## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start 

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

if the env file is config, those cmd will generate table if it not exists
